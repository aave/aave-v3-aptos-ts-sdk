Class UnderlyingTokensClient

The UnderlyingTokensClient class provides methods to interact with underlying tokens on the Aptos blockchain. It extends the AptosContractWrapperBaseClass and utilizes the TokensContract to perform various operations such as creating tokens, minting tokens, retrieving metadata, and more.

Param: provider

The provider instance for interacting with the Aptos blockchain.
